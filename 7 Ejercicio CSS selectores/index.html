Selectores CSS

Aplicando CSS a Diferentes Partes del HTML

1. El selector de elementos apunta a elementos basados en su nombre de etiqueta HTML.

  1. Los selectores de clase apuntan a elementos basados en el valor del atributo class.
  2. Los selectores de ID apuntan a elementos basados en el valor del atributo id.
  3. Los selectores de atributos apuntan a elementos basados en sus atributos y valores.
  4. El selector universal apunta a todos los elementos.
